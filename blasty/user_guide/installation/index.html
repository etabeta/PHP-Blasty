Installation Instructions

PHP Blasty is installed in three steps:

  1. Unzip the package.
  2. Upload the PHP Blasty folders and files to your server.
  3. Point your browser to PHP Blasty example page (usually http://localhost/blasty/examples/).

That's it!

If you're new to PHP Blasty, please read the Getting Started section of the User Guide to begin learning how to build PHP applications embedding YUI with PHP Blasty.
Enjoy!
