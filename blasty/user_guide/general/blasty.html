Blasty Class

The Blasty class permits to prepare the right environment for YUI components.

Preliminary steps

To get Blasty working in you application, You need to include the code and initialize it:

        ...
        require_once 'blasty/blasty.php';               // To gain blasty class in your php application
        $blasty = Blasty::getInstance();                // To instantiate the main blasty class
        ...

Loading a PHP Blasty component

The next step is to load PHP Blasty components with loadComponent() function.

        ...
        $calendar = $blasty->loadComponent('calendar'); // To load the calendar YUI widget
        ...

The function will return the name of the 'instance' (automaticcaly generated) for the loaded component.
It is possible to force the instance name.

        ...
        $calendar = $blasty->loadComponent(array('instance_name' => 'calendar')); // To load the calendar YUI widget
        ...

Alternative syntax is permitted, in this case the function will return an array of istances where the key is the instance name and the value is the component name:

  • Multiple component load with automatic instance name generation provided to the function as multiple function arguments
    •             ...
            $instances = $blasty->loadComponent('animation', 'calendar', 'autocomplete');
            ...
  • Multiple component load with automatic instance name generation provided to the function as list of component name
    •             ...
            $components_to_load = array('animation', 'calendar', 'autocomplete');
            $instances = $blasty->loadComponent($components_to_load);
            ...
  • Multiple component load with given instance name provided to the function as associative array where the keys are the instance names and the values are the component names
    •             ...
            $components_to_load = array('anim1' => 'animation', 'myCal' => 'calendar', 'ac_instance' => 'autocomplete');
            $instances = $blasty->loadComponent($components_to_load);
            ...

YUI components are automatic loaded by the PHP Blasty component loader, if you need to load only YUI component, please see Loading a YUI component page.
The last step is to Render pages with YUI components
See the PHP Blasty Widget Reference and PHP Blasty Utility Reference pages for more details.

Function Reference

blastyinfo()

blastyinfo()

This function, as phpinfo(), will show a summary with all info about PHP blasty.

Parameters

This function ha no parameter.

Applies to

This function is applicable only to Blasty (nested use).

Example

        ...
        $blasty->blastyinfo();
        ...

container()

container(string $component [, string $js_instance])

Return the HTML code necessary to contain the YUI component.

Parameters

$component
The component-name of the YUI component which container will be output.
$js_instance
The JavaScript instance of the YUI component which container will be output. Use this parameter if there are more YUI same components.

Applies to

This function is applicable to Blasty (nested use) or to each widget (stand-alone use) that have container. At the moment they are:
  • autocomplete
  • calendar
  • datatable

Example

        ...
        echo $blasty->container('calendar');
        ...
        echo $blasty->container('calendar', 'cal2');
        ...

loadComponent()

loadComponent(string $component [, string $component]) - loadComponent(array $component_array)

Load Blasty and YUI component, return their JS instances.
As explained above there are multiple syntax that can be used.

Parameters

$component
The component to load.
$component_array
The array of component to load as:
  • non-associative array where each element is the component to load
  • associative array where the key is the JS instance and the value is the component to load

Applies to

This function is applicable only to Blasty (nested use).

Example

        ...
        $instances = $blasty->loadComponent(array('anim1' => 'animation', 'myCal' => 'calendar', 'ac_instance' => 'autocomplete'));
        ...

setColumProperty()

setColumProperty(array $property)

Set column property for the datatable widget.
Multiple syntax can be used.

Parameters

$property
The property to set as a nested associative array where parent key is the column key (eg. id, amount, etc.) and the the child are couple of property => value (eg 'sortable' => true).

Applies to

This function is applicable to Blasty (nested use) only when loaded datatable widget or to datatable widget (stand-alone use).

Example

        ...
        $blasty->setColumnProperty(
          array(
              'id'       => array('label' => 'ID',     'sortable' => true,  'resizeable' => true),
              'amount'   => array('label' => 'Amount', 'sortable' => true,  'resizeable' => true),
              'quantity' => array('label' => 'Q.ty',   'sortable' => true,  'resizeable' => false),
              'title'    => array('label' => 'Title',  'sortable' => false, 'resizeable' => true),
               )
        );
        ...

yuiTags()

yuiTags([const YUI_CSS | YUI_JS])

Return the HTML code to embed YUI css and/or js.

Parameters

YUI_CSS
Return the HTML code to embed only YUI css.
YUI_JS
Return the HTML code to embed only YUI Java Script.

Applies to

This function is applicable to Blasty (nested use) or to each widget (stand-alone use).

Examples

        ...
        echo $blasty->yuiTags();
        ...
      
        ...
        echo $blasty->yuiTags(YUI_CSS);
        ...
        echo $blasty->yuiTags(YUI_JS);
        ...
