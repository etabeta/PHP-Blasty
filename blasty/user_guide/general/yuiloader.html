Loading a YUI component

With phpBlasty static class you have gained another php class that helps to embed YUI in HTML page in a correct way. It is YUILoader.
So phpBlasty has two method to load YUI:

  • The PHP way PHP generates HTML code to embed YUI
  • The JS way: Generation of JS code that will embed YUI
YUILoader class is automatically inizialized and used by PHP Blasty, but if you want you can freely use it with phpBlasty::yuiloader

Note: Be careful using this class to prevent unpredictable behaviour.

YUILoader: The PHP way

The phpBlasty class has the $phpblasty_config['yui_loader'] property that tells phpBlasty which loader method use between 'php' and 'js'. Default is 'php'.
With the PHP way, YUILoader will generate HTML code to embed YUI.

YUILoader: The JS way

If you changed the default loader method in 'js', YUILoader will generate JS code that will embed YUI in the page.

Load YUI components

To load one o more YUI component just use in your controller phpBlasty::yuiloader->load() function providing one component name for each function parameter

        ...
        phpBlasty::yuiloader->load('calendar', 'autocomplete');
        ...

Embed YUI in the HTML page

To embed YUI the HTML page just use in your view $this->ciwy->yuiTags() o $this->yuiloader->tags() function

        ...
        echo phpBlasty::yuiloader->yuiTags();
        ...
See the Render pages with YUI components page for more details.
