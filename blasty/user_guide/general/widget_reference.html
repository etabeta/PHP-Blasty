Common widget function reference

Blasty widgets have some functions identical, so these function are described here. If differences occurs, theese will be described in the widget function reference page.

container()

container([string $js_instance])

Return the HTML code necessary to contain the give instance of YUI component.
If no parameter given, the function will return the HTML code of the current JS instance.

Parameters

$js_instance
The JavaScript instance of YUI component which container will be output.

Example

        ...
        echo $widget->container('cal2');
        ...

generate()

generate()

Return the JS code necessary to get YUI work.

Parameters

This function ha no parameter

Example

        ...
        echo $widget->generate();
        ...

getContainerId()

getContainerId()

Return the id value of the css container used to contain the widget.

Parameters

This function ha no parameter

Example

        ...
        $container_id = $widget->getContainerId();
        ...

getCurrentInstance()

getCurrentInstance()

Return the JS current instance of the widget.

Parameters

This function ha no parameter

Example

        ...
        $js_instance = $widget->getCurrentInstance();
        ...

getInstance()

getInstance()

Return the object instance of the widget.
Since each widget is a singleton object, to instantiate it use this static function.

Parameters

This function ha no parameter

Example

        ...
        $widget = Calendar::getInstance();
        ...

setProperty()

setProperty(string $property, string $value) - setProperty(array $property_array)

Set property of the widget.
Multiple syntax can be used.

Parameters

$property
The property to set to.
$value
The value which the property is set.
$property_array
An associative array where key is the property and the value is ... the value.

Example

        ...
        $calendar->setProperty('PAGES', 3);                 // It will set the calendar property
        ...

yuiTags()

yuiTags([const YUI_CSS | YUI_JS])

Return the HTML code to embed YUI css and/or js.

Parameters

YUI_CSS
Return the HTML code to embed only YUI css.
YUI_JS
Return the HTML code to embed only YUI Java Script.

Applies to

This function is applicable only to Blasty.

Examples

        ...
        echo $blasty->yuiTags();
        ...
      
        ...
        echo $blasty->yuiTags(YUI_CSS);
        ...
        echo $blasty->yuiTags(YUI_JS);
        ...
