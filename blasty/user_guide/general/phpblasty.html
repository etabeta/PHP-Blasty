PHP Blasty static Class

Within Blasty there is a static class named phpBlasty used as common tools class.
Normally You do not need to know this class or use it in Your application, but if you want to extend or improve Blasty You need to read below.

Function Reference

flushYui()

Called from Blasty or each Blasty widget before call yuiTags() function to get sono YUI component loaded properly.

getBaseUrl()

PHP Blasty automatically detect base url to blasty folder and set blasty_url.

getPath()

PHP Blasty automatically detect the path to blasty folder and set the path to base_path, blasty, loader, component and language.

getVar()

To read the value of phpBlasty private var.

initYuiLoader()

Instantiate PHPLoader for Blasty or blasty widgets.

jsDataWrapper()

Return given data with JavaScript syntax.

loadLanguage()

Load the languae translation if necessary.

LogMessage()

Log Blasty messages (not yet developed). Work in progress.

setVar()

To set the value of phpBlasty private var.

setYuiBase()

Set the base of YUI library (local or remote).

showError()

Show Blasty error and exit.

showWarning()

Show Blasty error and continue.
